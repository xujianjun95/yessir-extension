YesSir 隐私政策

Chrome 扩展 · 最后更新：2026 年 4 月

核心承诺：YesSir 不收集、不上传、不共享任何用户数据。所有数据仅存储在您本地设备上，完全由您掌控。

我们收集哪些数据

YesSir 仅在您的本地设备上存储以下信息，用于实现扩展功能：

数据项 存储内容 用途
closeCount 历史累计操作次数（整数） 关闭标签 Toast 提示中显示累计次数
dailyStats 以日期为键的每日操作次数字典 悬浮窗中的近 5 日折线图统计
modifierKey 用户选择的修饰键名称（字符串） 记住用户对触发快捷键的自定义配置

权限使用说明

扩展申请了以下三项 Chrome 权限，均为功能所必需，不用于任何数据收集：

权限 使用原因
tabs 读取当前窗口的标签页列表，用于滚轮切换标签页浮层及双击关闭当前标签页功能
storage 将使用次数统计和修饰键配置保存至本地（chrome.storage.local），数据不离开设备
sessions 读取并恢复最近关闭的标签页，用于「后悔药」恢复功能

我们绝不会做的事

  • 不向任何服务器上传您的数据
  • 不收集您的浏览记录、标签页 URL 或页面内容
  • 不与任何第三方共享数据
  • 不使用任何广告 SDK 或分析追踪工具
  • 不在后台静默运行任何网络请求

您对数据的完全控制

所有本地数据均可随时清除：

  • 前往 chrome://extensions，找到 YesSir，点击「详情」→「清除数据」
  • 或直接卸载扩展，所有本地存储数据将随之完全删除

第三方服务

YesSir 不集成任何第三方服务、分析平台或广告网络。扩展的全部逻辑在您的本地浏览器中运行，不存在任何外部依赖。

未成年人保护

YesSir 为通用效率工具，不针对儿童设计，也不收集任何个人信息，因此不存在未成年人数据处理问题。

政策变更

如本政策发生实质性变更，我们将通过更新本页面的「最后更新」日期予以说明。建议您定期查阅。继续使用扩展即视为您接受最新政策。

联系我们

如对本隐私政策有任何疑问，欢迎通过 Chrome 应用商店的开发者联系渠道与我们沟通。

YesSir Chrome Extension 本政策适用版本：v1.3 及以上
